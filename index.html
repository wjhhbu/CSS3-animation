CSS3 动画属性如下：

@keyframes：

        定义动画：@keyframes  animationName {
          0 - 100%;
          from {} // 相当于 0%
          to {} 相当于 100%
        }

animation-name：动画的名称

animation-duration：完成一个动画的时间，例如 5s ， 5ms ；默认 0

animation-relay：动画延迟时间，例如 4s ，4ms

animation-direction：下一个动画播放的方向

normal：默认值，正常播放；
alternate：反向轮流播放

animation-play-state：动画播放的状态

running：播放；
paused：暂停

animation-timing-function：动画运动速度曲线

ease：默认，低速开始，然后加快，结束变慢
ease-in：低速开始
ease-out：低速结束
ease-in-out：低速开始和结束
linear：线性匀速

animation-fill-mode：动画前后的动画状态

none：默认值，原始状态
forwards：使用动画完成后的状态，即 最后一帧时的状态
backwards：使用动画前的状态， 即 第一帧的状态
both：使用动画前和动画后的状态，即第一帧和最后一帧的状态

animation-iteration-count：动画循环次数

默认值：1，
infinite：无限循环

animation：动画总简写

规则：name duration relay ...(其他属性值)；relay 的值 一定要放在 duration 后面
